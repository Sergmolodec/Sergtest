# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом его нужно обрамить звездочками (*). Например *вот так*

Чтобы выделить текст полужирным, его нужно обрамить двойными звездочками (**). Например **вот так**
